Freibier-Datenbankklassen zum objektorientierten Datenbankzugriff - alte Version. Diese Klassen sind im Rahmen des Freibier-Projektes entstanden und erlauben einen voll objektorientierten und von SQL abstrahierten Zugriff auf Datenbanken.

Die hier verwendete Bibliothek ist eine ältere Version, die inzwischen von Peter Hormanns stark überarbeitet in einer neuen Version in den Startlöchern steht. Da die neue Version aber eine Änderung der API mit sich bringt und noch nicht alle Funktionen der alten Version ausfüllt, ist hier vorerst noch diese im Einsatz. Aus diesem Grunde ist sie auch nicht so dokumentiert, wie das vielleicht sein sollte.
