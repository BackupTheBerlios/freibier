README zu templates/standard
----------------------------

Wird kein besonderes Theme angegeben, so wird "standard" als Theme
genommen. Dieses Theme kommt in dieses Verzeichnis. Allerdings enthält
es im Normalfall keine Template-Dateien, weil alle Templates, die
im Standard-Fall genommen werden sollen, die Default-Templates sind,
die im Dateibaum eine Ebene höher stehen.

Wer allerdings will, daß im Standard-Fall (kein Theme angegeben) andere
Templates verwendet werden, als bei allen anderen Themes, kann diese hier
hineinschreiben.
