README
------


Guten Tag, lieber Interessent für dieses hervorragende 
Softwarepaket (Hallo Peter!).

WebDatabase ist ein Framework für Webapplikationen im Tomcat, das bei
seinem Datenzugriff auf der Freibier-Datenbank-Klasse basiert. Es sind
drei einfache Applikationen verwirklicht, die als Beispiele dienen 
können.


Wer diese Datei hier gefunden hat, hat erstmal den Anfang gefunden,
von dem aus alle Fäden laufen. Wer mit der WebDatabase herumspielen
möchte bzw. sich darin umsehen möchte, sollte folgendes tun:

- Erzeugung von Javadocs per beiliegendem Ant-build-script 
  (Target javadoc)
- Erzeugung einer Distributions-Datei (*.jar) (Target dist)
- Die Javadocs sollten recht umfassend sein, also sich dort mal einlesen
  Was in den Javadocs fehlt oder unklar ist, bitte sofort melden (oder
  selbst verbessern).
- Im Projektwiki das HOWTO für neue Applikationen lesen
- Als Beispiel am besten erstmal mit dem DepotManager herumspielen.

Alle auftretenden Probleme, Schwierigkeiten, Konfigurationen etc.
unbedingt dokumentieren, damit die Nachwelt auch was davon hat. :-)
